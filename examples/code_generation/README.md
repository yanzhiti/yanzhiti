## 🎯 使用技巧

### 写好提示词

好的提示词包含:
1. **明确目标** - 要生成什么
2. **指定语言** - Python/JavaScript 等
3. **说明要求** - 性能、可读性等
4. **提供上下文** - 使用场景

### 示例对比

❌ **不好的提示词**:
```
写个排序
```

✅ **好的提示词**:
```
用 Python 实现快速排序算法，要求:
- 包含详细注释
- 添加类型提示
- 包含测试用例
- 时间复杂度 O(n log n)
```
